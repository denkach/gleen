# Gleen Motion Prototype v3

Эта сборка исправляет ситуацию, когда `http://localhost:8080/` продолжал открывать старый статический `index.html`.

## Самый простой запуск

### Windows
Дважды нажмите `START_WINDOWS.bat`.

### macOS / Linux
Откройте `START_MAC_LINUX.command` или выполните:

```bash
python3 start.py
```

Скрипт всегда запускает сервер из правильной папки, отключает кэш и открывает:

`http://localhost:8080/?motion=1`

В левом нижнем углу должна быть метка:

`MOTION ACTIVE · V3`

Если порт 8080 занят, остановите прежний сервер комбинацией `Ctrl+C` в старом терминале и запустите скрипт снова.

## Без сервера

Можно открыть `index.html` напрямую. Корневая страница теперь полностью автономна: CSS, JavaScript, SVG и motion-слой встроены внутрь.

## Управление движением

Нажмите на метку в левом нижнем углу, чтобы включить или уменьшить движение. Для прямого управления:

- `?motion=1` — принудительно показать анимации в прототипе;
- `?motion=0` — reduced-motion режим.

## Самопроверка

```bash
python3 start.py --check
```
